A Trading Platform

Company - FIS

Role - Automation Engineer

A product of FIS is a trading software working with the NYSE and NASDAQ exchanges, having over 170 business processes on its dashboard.

• Developed a Java-based Hybrid automation framework utilizing Selenium WebDriver, integrating data and keywords from xlsx files.

• Engineered Jenkins with Selenium framework to perform continuous testing, also deployment of iterative code changes.

• The Page Object Model (POM) design method was implemented to develop the framework, wherein web page objects were managed in distinct Java classes.
