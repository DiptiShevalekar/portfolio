Detect Cancer using ML

Several datasets such as symptoms of lung, breast, skin cancers were collected.

To work with image data Convolutional Nural Networks techniques for automatic feature extraction were used.

The model was trained and implemented using the preprocessed data, which includes splitting the data into training and testing sets.

The model's performance was evaluated using apropriate matrics such as precision, accuracy, recall.
